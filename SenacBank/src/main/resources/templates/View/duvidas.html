Dúvidas Frequentes

Veja respostas para as perguntas mais comuns

Como faço para depositar dinheiro?

Para depositar, clique no botão "DEPOSITAR" na página inicial, informe o valor e confirme a operação.

Como posso sacar meu saldo?

Vá até a opção "SACAR", informe o valor desejado e siga as instruções para concluir o saque.

Como transferir para outro usuário?

Selecione "TRANSFERIR", informe o ID do destinatário e o valor, depois confirme a transferência.

O que é a Caixinha?

A Caixinha é uma funcionalidade para guardar dinheiro separado do saldo principal, ideal para economias.

Como vejo meu extrato?

Clique em "Ver Extrato" na página inicial para visualizar todas as suas transações.

Esqueci minha senha, o que faço?

Entre em contato com o suporte para redefinir sua senha.
